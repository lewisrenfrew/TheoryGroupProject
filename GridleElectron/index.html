Gridle

Welcome to Gridle a tool for simulating Potential and Electric Fields in (charge-free) electrostatic problems. If this is your first time using Gridle then it is recommended to Preprocess, Simulate and then View your system using the buttons below. For more information about the data input format and the information that needs to be provided please press the User Manual button. Please scroll down for creator and attribution information.
